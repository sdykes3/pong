This game is basically Pong.
Player 1 controls: A button (Z on PC), B button (X)
Player 2 controls: up arrow, down arrow

Press Select (backspace) to reset the game at any time.

Lose condition is if the ball starts to go past either 
paddle, just press Select to restart. Win condition is 
when 99 points are scored. The gameplay is cooperative 
rather than competitive in this case.
